# Shoiko
Rear mounting of electrical components mod for VORON2.4

## What is Shoiko?
Shoiko(背負子)  is an Japanese word meaning "backpack". (naming inspired from [Kirigami](https://github.com/christophmuellerorg/voron_0_kirigami_bed).)
